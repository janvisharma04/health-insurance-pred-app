🧠About the Project

This web application predicts medical insurance costs using advanced machine learning models trained on real-world insurance data. Our system analyzes multiple personal and health factors to provide accurate cost estimates.

  • 🧪Model: XGBoost, Gradient Boosting, Random Forest
  • 📁Dataset: Medical Insurance Cost Dataset (3000 records)
  • ⚙️Features: Age, BMI, Smoking Status, Region, Children count
  • 📊Accuracy: 95% accuracy with Random Forest
